# 久安问道录 Demo 玩法实验记录 - 首轮差异验证

## 1. 本轮目标

本轮不是做精确平衡，而是先验证一件事：**不同“选课方向 + 自由时段风格”组合，是否已经能稳定拉开不同的一周打法。**

因此，这一轮采用的是组合验证，而不是单变量对照：

- A 组：偏学习推进
- B 组：偏工坊 / 打工推进
- C 组：偏修炼 / 养神推进

如果这三组最后仍然高度趋同，就说明当前 demo 的 build 差异还不够强。

## 2. 控制条件

- 代码版本：`d273365`
- 运行方式：本地静态服务 + Playwright 半自动试跑
- 随机种子：固定为运行时默认种子 `20260313`
- 随机事件：开启
- 夜间记忆：采用半自动贪心策略
  - 先放灵台锚片，优先开启灵台核心与当前路线主分区
  - 其次优先拼出与路线匹配的建筑
  - 能连桥时优先补桥
- 说明：
  - 因为夜间构筑不是“最优解搜索”，所以本轮更适合看趋势，不适合拿来做极限数值结论

## 5. 关键观察

### 5.1 A 学究推进组

- 第 1 天：
  - 固定课是《天道算法导论》
  - 夜间直接析出 `结构纹片 2 + 回响纹片 2 + 养神纹片 2 + 专注纹片·数术 1`
  - 当晚先在灵台核心定型 1 座悟理阁
- 第 3 天：
  - 数术已经抬到 9，道法已经到 12
  - 夜间开始在数术区稳定长出悟理阁，说明“课程 + 课业”组合已经能稳定导向学习构筑
- 第 5 天：
  - 《阵法拓扑结构》开始显著抬升阵法相关碎片，阵法技能开始追上来
  - 但夜间主要建筑仍偏数术 / 道法，没有明显把符法区或阵法区进一步拉开
- 第 7 天：
  - 最终形成的是典型“高悟道点、低灵石、课程沉淀极强”的路线
  - `数术 12 + 道法 12 + 阵法 11` 的技能形态已经和另外两组完全不同

结论：

- 学习流差异已经成立，而且非常明显
- 课业行为对悟道点和技能沉淀的放大作用很强
- 但目前学习流的综合收益仍偏高，已经接近“除了灵石之外都很强”

### 5.2 B 工坊行务组

- 第 1 天：
  - 第一天结束灵石已经冲到 36，悟道点只有 3
  - 夜间主要是核心悟理阁，工坊方向没有第一时间爆发
- 第 3 天：
  - 炼器区终于开始长出术式楼，炼器主线成型
  - 这时灵石来到 62，辅导员关系也在同步上涨
- 第 5 天：
  - 《精准炼丹学》开始把灵物方向拉出来，路线从“纯打工”变成“炼器 + 灵物 + 符法”的实作型 build
  - 但悟道点仍明显落后
- 第 7 天：
  - 最终 `灵石 99`，几乎顶格
  - `炼器 9、符法 8、灵物 4`，方向非常清楚
  - 同时 `疲惫 10`，说明这套路线的代价已经显性存在，但惩罚还不够重

结论：

- 工坊 / 打工流也已经成型，资源轮廓与学习流明显不同
- 灵石、辅导员关系、炼器成长都明显更高
- 问题在于：疲惫虽然拉满了，但没有真正把这条路线压出“危险感”

### 5.3 C 修体养神组

- 第 1 天：
  - 第一天夜里就出现 `专注纹片·道法 2 + 养神纹片 2`
  - 直接在道法区与灵台核心定型出 `术式楼 + 养神台`
- 第 3 天：
  - 一次性开出多个道法区节点，并连续补出道法区建筑
  - 这条路线的夜间构筑反馈最直观，玩家很容易感受到“今天练功确实有在养夜修”
- 第 5 天：
  - 《经络解剖学》带出灵物方向，但整条线仍明显以道法区为核心
  - 形成“恢复 / 修炼 / 少量学理”的稳定 build
- 第 7 天：
  - 最终 `道法 11、灵力 10`
  - 道法区建筑数量明显最多，夜间建筑辨识度强

结论：

- 修炼 / 养神流的玩法识别度很高，尤其体现在夜间养神台和道法区建筑的堆叠上
- 它不像学习流那样堆悟道点，也不像打工流那样堆灵石，但 build 感很清楚
- 问题是：这条路线的疲惫也能堆得很高，说明“持续修炼”的损耗反馈还可以再强化

## 6. 首轮结论

### 已经验证成立的点

1. 不同组合路线已经能明显拉开结果。
   - A 组拿到高悟道点和高学术技能
   - B 组拿到极高灵石和更强的炼器 / 符法 / 灵物
   - C 组拿到更强的道法区构筑和更高灵力感

2. 白天行为已经会影响夜间灵块分布。
   - 学习流更容易出现结构纹片、回响纹片
   - 打工 / 行务流更容易出现衔接纹片、恢复向碎片，且课业回响明显减少
   - 修炼流更容易堆出养神纹片和道法向专注纹片

3. 夜间建筑已经能反向体现 build。
   - A 组偏悟理阁、术式楼
   - B 组偏炼器区术式楼与少量养神台
   - C 组明显偏道法区术式楼与养神台

### 仍然暴露的问题

1. 三组最后都拿到“上上品”，说明周结算评级公式区分度不足。
   - 当前评级还不能有效放大路线差异
   - 更像是“都能赢，但赢法不同”，而不是“不同路线会得到不同层级结果”

2. 疲惫代价已经出现，但压迫感还不够。
   - B 组疲惫 10 仍能高分收官
   - C 组疲惫 10 也没有明显把效率压下来
   - 这会让高强度路线缺少真正的取舍压力

3. 学习流综合收益偏强。
   - A 组除了灵石明显偏低，其余几乎全面领先
   - 说明“做课业”的沉淀收益和课程收益叠加后，可能已经过于稳定

4. 选修课对夜间分区的专属牵引还可以更强。
   - 例如 A 组选了《阵法拓扑结构》，阵法技能最终确实升高，但夜间主建筑仍没有更强地压向符法区 / 阵法区
   - 这说明“课程方向 -> 灵块分区 -> 建筑分区”的链路还可以再拉开

## 7. 建议的下一轮实验

### 下一轮优先做什么

建议下一轮不要继续同时改“原型 + 选课 + 日程风格”三个变量，而是拆开验证：

1. 固定同一原型与同一选课，只改日程风格
   - 验证“做课业 / 去打工 / 去修炼”本身谁更强、谁更稳、谁更亏

2. 固定同一日程模板，只改选修课
   - 验证选修课到底能不能把夜间灵块与最终建筑真正拉向不同分区

3. 在第二轮里重点盯 3 个指标
   - 疲惫是否真的会压缩高强度路线收益
   - 课业是否已经成为默认强选
   - 周结算评级是否需要重新设计区分逻辑

### 调参方向

- 提高高疲惫对课业、打工、修炼收益的压制
- 让灵石、悟道点、灵力值在评级里形成更明确的权重冲突
- 强化选修课对夜间碎片分区与专注方向的影响
- 适度下调“连续做课业”的综合上限，避免学习流过于全面

## 8. 一句话结论

当前 demo 已经初步具备“同一周流程下打出明显不同 build”的能力，可以进入第二轮更细的单变量验证；但**评级、疲惫惩罚、选修课分区牵引**这三块，已经成为下一步最值得优先调的地方。
